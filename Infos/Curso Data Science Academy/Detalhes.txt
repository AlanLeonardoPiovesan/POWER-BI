-- MODELO DE ORGANIZAÇÃO DE DADOS
1º PRECISAMOS CARREGAR OS DADOS DE UMA FORMA QUE FACILITE A EXTRAÇÃO DE RELATÓRIOS
2º ESCOLHER O MELHOR MODELO DE ORGANIZAÇÃO DE DADOS
	-PROCESSOS DE NEGÓCIO: IDENTIFICAR, COMPREENDER E DESCREVER - EX: VENDA DE PRODUTOS ELETRONICOS
	-DEFINIR O NÍVEL DE DETALHAMENTO (GRANULARIDADE) - EX: TOTAL DE PRODUTOS MAIS VENDIDOS POR DIA
	-IDENTIFICAR AS DIMENSÕES NECESSÁRIAS: DEFINIR O QUE PRECISA SER VISTO E DA MELHOR FORMA - FABRICANTE, PRODUTO, LOJA, TEMPO
	-CRIAR SCHEMA: MODELO IMPLEMENTADO NO BANCO DE DADOS PARA ORGANIZAÇÃO DOS DADOS - MODELO CRIADO NO DW OU POWER BI

-- GRANULARIDADE= DIFERENTES NIVEIS DE DETALHAMENTO DOS DADOS

- O QUE É DASHBOARD?
É uma ferramente de gestão e visualização de informações que é usado para monitorar indicadores de desempenho (KPI's), métricas e outros pontos de dados relevantes para o negócio, departamento ou projeto. Uma ferramenta de gerenciamento de informações.
